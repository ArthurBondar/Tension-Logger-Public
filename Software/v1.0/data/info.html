Settings

Date and Time [selection] Date and time of the internal clock chip
Graph points [0 to 100] Number of datapoints plotted on the graph on the graph at once (0 = disable graph)
Refresh rate [1 to 1800] Rate at which new datapoints are displayed on the graph
Setpoint [any] Maximum load indicator
Logging interval [1 to 1800] Rate at which the data is stored to the SD card in seconds
Coin Cell [V] Voltage of the coin cell battery
Temperature [°C] Temperature inside the enclosure

Status

normal operation reading from sensor and writing to SD card
storage card not found SD card not detected
sensor not connected no incomming data from the tension sensor
coincell battery low voltage on the coincell battery is below threshold
internal error any internal system error
sensor parsing error failed to parse data from the tension sensor
memory overflow internal memory overflow in the unit - reboot required
